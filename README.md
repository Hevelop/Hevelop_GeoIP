Hevelop GeoIP for Magento 2.x
=====================

This module is a Magento 2 porting of [Sandfox_GeoIP](https://github.com/magento-hackathon/Sandfox_GeoIP).

Requirements
------------
- PHP >= 5.5.0

Compatibility
-------------
- Magento >= 2.0;
